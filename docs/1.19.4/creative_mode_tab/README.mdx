[创造模式物品栏](https://minecraft.fandom.com/zh/wiki/%E7%89%A9%E5%93%81%E6%A0%8F#%E5%88%9B%E9%80%A0%E6%A8%A1%E5%BC%8F%E7%89%A9%E5%93%81%E6%A0%8F)是一些物品的集合，并且如他的名字一样只在创造模式出现。

曾开发过以前版本的 Mod 开发者可能意识到了：`Item.Properties` 里的 `tab` 不见了。

在 1.19，tab 不再是 `Item` 侧进行设置，改为了使用 builder 的方式构建一个 `CreativeModeTab`：

<Tabs>
<TabItem value="core" label="核心代码">

```java title="src/main/java/org/teacon/xiaozhong/Xiaozhong.java"
// 此处仅展示如何添加物品，builder 内有更多方法需要读者自行查看
public static void registerCreativeModeTab(CreativeModeTabEvent.Register event) {
    event.registerCreativeModeTab(new ResourceLocation("xiaozhong", "main_tab"), builder ->
            builder.displayItems((parameters, output) -> {
                output.accept(SULFUR_DUST_ITEM.get());
                output.accept(SULFUR_BLOCK_ITEM.get());
            }));
}
```
